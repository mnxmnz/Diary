## 🖥 Local 실행 방법

#### [1] NPM & Node.js 설치

[NPM & Node.js 설치 바로가기](https://www.npmjs.com/get-npm)

#### [2] Clone the Repo

```sh
git clone https://github.com/mnxmnz/Diary.git
```

```sh
cd Diary
```

#### [3] Install Project Packages

```sh
yarn
```

#### [4] Run the Project

```sh
yarn start
```
